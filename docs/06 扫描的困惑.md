扫描文件夹时，`PlayerService.cs:191~258` 的 `CreateTrackFromFile()` 方法负责计算 `TotalSamples`，有两种方式：

**① TagLib 估算（默认用于普通单文件）：**

```
long samplesA = (long)(audioFile.Properties.AudioSampleRate * audioFile.Properties.Duration.TotalSeconds);
```



**② NAudio 精确计算（A/B 融合文件使用，也用于普通文件的回退）：**

```
long totalSamples = reader.Length / reader.WaveFormat.BlockAlign;
```
